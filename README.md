# Tweeter Project

Tweeter is a simple AJAX based twitter clone that uses jQuery, HTML5, and plain old CSS3. 
This is an assignment for Lighthouse Labs with the goal of learning these languages.

## Dependencies

- Express
- Node 5.10.x or above
- Moment.js
